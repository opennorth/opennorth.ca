Job summary

Position title: IT Suport Technician Salary scale: $20.51-$24.87/hr (40,000-48,500 annual equivalent)

Reporting to the Operations Manager, in collaboration with the Applied Research Lead, the incumbent will be the principal IT support for the Company. Key responsibilities will include diagnosing hardware and software issues, user provisioning and training, website updates, and information management, implementation and maintenance of security practices; within a cloud-centric, serverless environment.

OpenNorth

Founded in 2011, OpenNorth is Canada’s leading not-for-profit organization specialized in open data, open government, community engagement, open smart cities and civic technology. We work with Canada’s largest, most innovative, and connected cities. Our network extends across the country and globally through communities of practice and open government partners. Our work promotes the effective use of data as infrastructure facilitating sustained engagement between government, civil society, and the public.
OpenNorth's computing environment is lean, cloud-based, with 15-20 staff.

Major Responsabilities

  • Maintain Company software and subscriptions
  • Maintain Company data backups
  • Maintain Company hardware
  • Troubleshoot and resolve various technical issues via phone, email
  • Develop documentation/user guides for common user issues
  • Performan on/of staff with respect to information management and information security practices, access rights, and use of the software.
  • Maintenance of OpenNorth's Represent API
  • OpenNorth website maintenance

General and Specific Duties

  • Organizational information practices
    • Make recommendations on the lifecycle of IT technologies and processes
    • Outline an organization-wide strategy for OpenNorth to backup their critical data
      • Review the various backup options and technologies issues that organizations should consider when implementing a backup plan
      • Choice of backup system matters and needs to be in line with different data retention and redundancy needs (can differ based on project/program)
      • Set up and perform regular backups of all Company data to a third location, either on-site or off-site
    • Contribute to implementing formalized processes to help ongoing and future IT team operations
  • Documentation
    • Create documentation to describe each service and facilitate access (including purpose, methods of access, installation)
  • User provisioning (hardware and software)
    • Setup new hardware and users, devices, and provide access to internal applications; install software;
  • Hardware and software procurement and maintenance
    • Evaluate and provide recommendations on procurement requests based on features/functionality, price, minimum required specifications
    • Provide support on software installations and updates as needed
    • Ensure urgent security patches are installed across Company hardware
    • Ensure operating system and firmware updates are installed on a timely basis
    • Install and configure computer hardware, software, systems, networks, printers, and scanners
  • Maintain hardware and software equipment inventory
    • Label and inventory all hardware
    • Maintain current knowledge of all software licenses and online subscriptions owned by the Company including current status, maximum number of installations, current installations, expiration/billing renewal dates
  • Represent API
    • Code maintenance - support maintenance of Python scrapers
    • Data updates - support manual updates of key datasets (e.g. candidate lists during elections)
    • Coordinate standard responses with external collaborators and email responses to information enquiries from users of Represent
  • Represent APi server management
    • Manage server-side issues related to Represent API
  • OpenNorth website
    • Website content updates
    • Edit OpenNorth website templates and structure
    • Edit Represent (represent.opennorth.ca) website content
